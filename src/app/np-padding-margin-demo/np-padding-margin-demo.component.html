Padding and Margin

Padding and Margin can be applied as {property}{sides}-{size}

Where property means:

  • m - for classes that set margin
  • p - for classes that set padding

Where sides means:

If sides are not mentioned then all sides will be included
  • t - for top
  • r - for right
  • b - for bottom
  • l - for left

Where size means:

Here $spacer is SASS variable declared in theming
  • 1 - for $spacer/2
  • 2 - for $spacer
  • 3 - for $spacer*2
  • 4 - for $spacer*3
  • 5 - for $spacer*4

Examples:

  • m-1
  • mt-2
  • mr-3
  • pb-4
  • pl-5

Auto Margin

  • use m-auto for all side auto margin
  • use mt-auto for top auto margin
  • use mr-auto for right auto margin
  • use mb-auto for bottom auto margin
  • use ml-auto for left auto margin
