NpDialogService

Inject NpDialogService into constructor of component

 How to open dialog?
open(content: string | TemplateRef<any>, config: NpDialogConfig): NpDialogRef;

Here open method has 3 parameters, and this method return the overlay reference object of class NpDialogRef

Parameter Description Type
content Pass content of dialog
Here for dialogTemplate, let-data is available, where data is value passed in open method. 		string | TemplateRef<any>
config Pass dialog config with below properties.
Property Description Type Default
type Pass any of "alert" | "confirm" | "prompt" string "alert"
hasBackDrop Has backdrop or not boolean true
backDropClass Backdrop class name string np-dialog-backdrop
styleClass Set extra class on component string
inputId Set id attribute string
NpDialogConfig
data Pass data to dialog, for template this data is available as context. any
How to subscribe onClose method of dialog?

open method returns overlay reference object, which has property onClose: Subject<any>
