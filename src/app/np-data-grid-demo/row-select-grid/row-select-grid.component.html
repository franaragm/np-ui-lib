Row selection example

To set single row selection, set singleRowSelectEnable to "true" in grid definition

To set multi row selection, set multiRowSelectEnable to "true" in grid definition

For select all, in server side operations only those rows will be selected which are fetched through API using pagination, in client side all rows will be selected.

Set selectRowOnClick to "true" for selecting row on row click.
