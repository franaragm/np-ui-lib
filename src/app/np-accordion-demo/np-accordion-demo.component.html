<np-accordion>
Property Description Type Default
defaultOpenIndex Set which panel will be default open at load time number
allowMultipleOpen Allow multiple panels to be open, if set to false, then when open new panel, other panels will be close boolean false
styleClass Set extra class on component string
inputId Set id attribute string
<np-accordion-item>
Property Description Type Default
title Title of panel string | TemplateRef <any>
isOpen Set default open boolean false
height Set height for accordion item in px number
disabled Set disabled boolean false
iconCss Set CSS for icon like 'fa fa-home' string
styleClass Set extra class on component string
inputId Set id attribute string
<np-accordion>
Method Description Parameters
onExpand($event) Callback on expanding panel. $event
onCollapse($event) Callback on collapsing panel. $event
<np-accordion>
Api Description Parameters
expandByIndex(idx: number) Expand by index, start with 0. idx: number
expandById(id: string) Expand by inputId of panel. id: string
collapseByIndex(idx: number) Collapse by index, start with 0. idx: number
collapseById(id: string) Collapse by inputId of panel. id: string
