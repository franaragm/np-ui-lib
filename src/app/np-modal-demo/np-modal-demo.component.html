Modal

Module to import
NpModalService

Inject NpModalService into constructor of component

 How to open modal?
open(content: string | TemplateRef<any> | Type<any>, config: NpModalConfig, data: any): NpModalRef;

Here open method has 3 parameters, and this method return the overlay reference object of class NpModalRef

Parameter Description Type
content Pass content of modal
Here for modalTemplate, let-data and let-close are available, where data is value passed in open method, and close is a method to close modal.
string | TemplateRef<any> | Type<any>
config Pass modal config with below properties.
Property Description Type Default
header Header text string
height Set height. Values should be like '100%', '100px', 200. If pass as number then it will considered as px. number | string
width Set width. Values should be like '100%', '100px', 200. If pass as number then it will considered as px. number | string
hasBackDrop Has backdrop or not boolean true
backDropClass Backdrop class name string np-modal-backdrop
showCloseButton Show close button or not boolean true
closeOnClickOutside Close modal on outside click or not boolean true
styleClass Set extra class on component string
inputId Set id attribute string
NpModalConfig
data Pass data to dialog, for template this data is available as context. any
How to subscribe onClose method of modal?

open method returns overlay reference object, which has property onClose: Subject<any>

How to access data and NpModalRef object in another component?

In another component, overlay reference is injected using injector.

Here ref object contains data: any and close(data: any) method. Which will close overlay.

Pass response data in close(data: any) method's parameter, which will be accessible where onClose is subscribed.

Use overlay reference's close() method to close modal from another component.
